# 4Key Rhythm Game

4키 리듬게임입니다. 떨어지는 노트를 타이밍에 맞춰 키를 눌러 점수를 획득하세요!

## 설치 방법

```bash
npm install
```

## 실행 방법

```bash
npm run dev
```

브라우저에서 `http://localhost:5173` (또는 표시된 포트)로 접속하세요.

## 조작 방법

- **D** 키 또는 **←** (왼쪽 화살표): 첫 번째 레인
- **F** 키 또는 **↓** (아래 화살표): 두 번째 레인
- **J** 키 또는 **↑** (위 화살표): 세 번째 레인
- **K** 키 또는 **→** (오른쪽 화살표): 네 번째 레인

## 게임 규칙

- 노트가 판정선(빨간 선)에 도달했을 때 해당 레인의 키를 누르세요
- 판정 등급:
  - **PERFECT**: ±50ms 이내 (100점)
  - **GREAT**: ±100ms 이내 (80점)
  - **GOOD**: ±150ms 이내 (50점)
  - **MISS**: 타이밍 실패 또는 노트를 놓침 (콤보 끊김)

## 기능

- 4개의 레인에서 노트가 떨어지는 리듬게임
- 실시간 점수 및 콤보 표시
- 정확도 계산
- 판정 피드백 애니메이션
- 30초 게임 플레이

## 기술 스택

- React 18
- TypeScript
- Vite
- CSS (인라인 스타일)
